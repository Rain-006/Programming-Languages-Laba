Делегирование событий

Один обработчик для группы элементов. Событие всплывает и перехватывается общим обработчиком.

classList

Удобный API для работы с классами: add(), remove(), toggle(), contains().

add()

Добавляет класс к элементу. Можно добавить несколько через запятую.

remove()

Удаляет указанный класс. Безопасно — если класса нет, ошибок не будет.

contains()

Проверяет наличие класса. Возвращает true/false.
